


# Summary

This assignment is a single screen app which shows the top rated Movies fetched from a TMDB API with configureable theme modes using followings. 

 - Kotlin
 - MVVM
 - LiveData
 - Hilt
 - Shimmer view
 - Lottie animation
 - Glide
 - Retrofit
 - Coroutines
